# GreenOnions.TicTacToe
[GreenOnions](https://github.com/Alex1911-Jiang/GreenOnions)的井字棋游戏插件

功能说明：可通过QQ消息进行井字棋人机对战游戏，操作方法支持表情涂鸦和发送坐标
